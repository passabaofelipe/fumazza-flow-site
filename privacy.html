Privacy Policy

Last updated: April 2026

1. Overview

Fumazza Flow ("the App") is a local desktop application. It does not operate any cloud servers or databases on behalf of users. All processing occurs on the user's own machine.

2. Data We Collect

The App does not collect, transmit, or store any personal data on external servers. The following data is stored locally on the user's device only:

  • TikTok OAuth access tokens and refresh tokens (stored in local files)
  • Channel and video metadata entered by the user
  • Publish job history (stored in a local MongoDB instance)

3. TikTok Data Usage

When a user authenticates with TikTok, the App receives and locally stores:

  • open_id — used to identify the TikTok account
  • display_name — shown in the channel configuration UI
  • access_token / refresh_token — used to authenticate API calls for uploading and publishing videos

This data is never sent to any third party or external server controlled by Fumazza Flow.

4. How We Use TikTok API

The App uses the TikTok Content Posting API exclusively to upload and publish videos that the user explicitly selects. No content is posted without the user's direct action (manual publish or pre-configured schedule set by the user).

5. Data Retention

All data is stored on the user's local machine. Users can delete their credentials and data at any time by removing the local credential files or uninstalling the App.

6. Third-Party Services

The App communicates directly with the following official APIs only:

  • TikTok API (api.tiktok.com)
  • YouTube Data API v3 (googleapis.com)
